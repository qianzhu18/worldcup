**更新日期：** 2026-06-13

---

## 一、产品定位

**JMWL 不是押注平台，也不只是赛程表，而是一个"AI 看球搭子"。**

Slogan：**"赛前有底，赛后有谱"**

核心链路：赛前策略 → 模拟采纳 → 赛后复盘 → 用户画像

## 三、一个月执行计划

### Week 1（6/13-6/19）：核心玩法闭合

**目标：** 用户能完成"预测 → 赛后结算 → 积分变化 → 排行榜"完整循环

| 天 | 任务 | 产出 |
|----|------|------|
| D1 | 每日签到 UI + 预测押注滑块 | 用户可领积分、押注预测 |
| D2 | 赛后自动结算定时任务 | 比赛结束后自动结算积分 |
| D3 | 排行榜数据验证 + "我的排名" | 排行榜有真实数据 |
| D4 | 成就系统 UI + 解锁 toast | 成就徽章展示 |
| D5 | 兑换商城页面 | 积分可兑换商品 |
| D6 | AI 准确率仪表盘 | 建立信任的关键页面 |
| D7 | 集成测试 + Bug 修复 | 全链路跑通 |

**验收标准：** 用户注册 → 领积分 → 预测 → 赛后看积分变化 → 上排行榜

---

### Week 2（6/20-6/26）：社交传播和增长

**目标：** 用户有动力把产品分享出去

| 天 | 任务 | 产出 |
|----|------|------|
| D1-2 | 预测结果分享卡片（@vercel/og） | 赛后生成可分享的 OG 图 |
| D3 | 邀请返积分机制 | 注册送 200 积分，邀请人也得 200 |
| D4 | 每日精选 Top 5 页面 | AI 推荐当日最有价值比赛 |
| D5 | SEO meta 标签 + 结构化数据 | 搜索引擎可抓取 |
| D6-7 | 社区入口 + 推广物料 | 微信群/Discord 入口 |

**验收标准：** 用户可生成分享卡 → 朋友通过分享链接注册 → 双方获得积分

---

### Week 3（6/27-7/3）：变现和赞助

**目标：** 验证商业模式，接触赞助商

| 天 | 任务 | 产出 |
